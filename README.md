## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Build for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```

### Seed

```sh
npm run db:seed:dev # massive fake data
or
npm run db:seed:prod # less data, but like real
```
